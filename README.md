# debian-game-servers

Basic setups for our Debian-based game servers.

## Usage

Initial setup:

```
sudo apt install make && git clone https://github.com/canine-systems/debian-game-server.git && make install
```

Build the .deb:

```
make
```

Build + install the .deb:

```
make install
```
